# Bit-Manipulation

This repository houses bit manipulation functions. This exercise was an assignment in my Computer System class. See bits.h for detailed explanation of functions.

#### Functions:
* bitXor
* thirdBits
* upperBits
* byteSwap
* copyLSB
* divpwr2
* isEqual
* isLess
* rempwr2
* satMul2
* bang
* reverseBits
* ilog2
